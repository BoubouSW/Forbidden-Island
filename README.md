## <br> **Commands**
<br>

- `Z` - Up
- `Q` - Right
- `S` - Down
- `D` - Left
- `F` - Dry Mode
- `R` - Catch Artifact
- `V` - Cards Trade
- `M` - Cards Trade with everybody (Messenger only)
- `T` - Throw away a card
- `L` - Use Sandbag
- `H` - Use Helicopter
- `P` - Fly Mode (Pilot Only)
- `N` - Navigate mode (Navigator only)
- `A` - Up Left (Explorer only)
- `E` - Up Right (Explorer only)
- `W` - Down Left (Explorer only)
- `C` - Down Right (Explorer only)
